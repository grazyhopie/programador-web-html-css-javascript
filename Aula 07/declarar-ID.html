ID em CSS

Como agrupar regras para vários elementos diferentes.

Os ID sao definidas no CSS com nomes iniciados pela # (hastag). As regras são indicadas pelos atributos. Os elementos no HTML que usarão estas regras deverão ter o atributo ID="nome_do_ID{propriedade: valor;}".
