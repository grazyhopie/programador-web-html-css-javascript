Classes em CSS

Como agrupar regras para vários elementos diferentes.

As classes sao definidas no CSS comm nomes iniciados pelo . (ponto). As regras são indicadas pelos atributos. Os elementos no HTML que usarão estas regras deverão ter o atributo class="nome_da_classe{propriedade: valor;}".
