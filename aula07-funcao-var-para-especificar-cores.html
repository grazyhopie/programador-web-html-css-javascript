Como usar a função var()

A função var() pode ser usada para definir cores com codigos(rgb(num1, num2, num3), #1e90ff) em variveis especificas. Você deve criar uma classe escrita ":root" e as propriedades irão ser as variaveis que vc vai armazenar o nome das cores. Essas variaveis precisam começar com "--". nas proximas classes ou id que vc declarar é só colocar: propriedade: var(--nome_da_cor);
